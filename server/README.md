# Procedural WebFPS Server

Install & run:

1. `cd server`
2. `npm install`
3. `node server.js`

The server listens on port 8080 by default. It accepts WebSocket clients and runs a simple authoritative tick loop.

Note: This demo server is intentionally compact and meant for local testing/simulation. It provides basic state sync and input validation scaffolding used by the client demo.
